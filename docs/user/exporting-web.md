# Exporting Web Images

Use PNG export to create web-ready image derivatives from attached source images.

OA Curator provides two PNG export presets:

- **Basic:** resizes images taller than 800px down to 800px high.
- **Premium:** resizes images taller than 2000px down to 2000px high.

The generated PNG is recorded with the Artwork as a derivative. OA Curator does not overwrite the original source image.
