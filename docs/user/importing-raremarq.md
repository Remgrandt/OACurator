# Importing From Raremarq

Raremarq currently does not provide a bulk export file.

OA Curator can still store Raremarq URLs and gallery-site IDs on Collections, Galleries, and Artworks.

If you want Raremarq to support collection export, use the [Raremarq feature suggestion form](https://forms.gle/ri5ATNyqKCUkG8iU9). OAA is the preferred export format to request because it can carry Collection, Gallery, Artwork, gallery-site metadata, and optional embedded files in a single collector-controlled archive.

Raremarq terms are at [https://www.raremarq.com/terms-and-conditions](https://www.raremarq.com/terms-and-conditions) and prohibit scraping.
